# Task 1: Greeting Script

## Description
Create a script that takes a user's name as an argument and prints:
```
Hello, <name>! Welcome to Bash scripting.
```

If no name is provided, print:
```
Hello! You didn't provide a name.
```

## Usage
```bash
./greeting.sh John
# Output: Hello, John! Welcome to Bash scripting.

./greeting.sh
# Output: Hello! You didn't provide a name.
```

## Concepts Covered
- Command-line arguments (`$1`)
- Conditional statements (`if-else`)
- String testing (`[[ -n ]]`)
