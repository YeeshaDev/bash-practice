# Task 2: Number Check

## Description
Write a script that takes a number as an argument and prints whether it is positive, negative, or zero.

## Expected Output
```bash
./number_check.sh 5
# Output: 5 is a positive number

./number_check.sh -3
# Output: -3 is a negative number

./number_check.sh 0
# Output: Number is zero

./number_check.sh
# Output: Provide a number
```

## Concepts Covered
- Command-line arguments
- Multiple conditional statements (if-elif-else)
- Numeric comparisons (`-lt`, `-gt`)
- String testing (`-z`)
